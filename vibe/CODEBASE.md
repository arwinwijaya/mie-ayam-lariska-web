## 6. Data Model

### Firebase Realtime Database Structure

```
menu/
├── {item_id}: {
│     name: string,
│     category: "Mie Ayam" | "Minuman" | "Topping Tambahan",
│     price: number (Rupiah),
│     description: string (max 200 chars),
│     status: "available" | "limited" | "sold_out",
│     badge: "favorit" | "baru" | "bestseller" | "populer" | null,
│     order: number
│   }

packages/
├── {package_id}: {
│     name: string,
│     description: string (max 300 chars),
│     icon: string (emoji),
│     items: string[] (menu item IDs),
│     price: number (Rupiah),
│     tag: "Basic" | "Best Seller" | "Puas" | "Spesial",
│     isFeatured: boolean,
│     isActive: boolean,
│     whatsappMessage: string,
│     order: number
│   }

stock/ (backward compatibility alias)
├── {item_id}: { status: "available" | "limited" | "sold_out" }
```

### Menu Items (24 items)
- **Mie Ayam** (11 items): Mie Ayam Mini, Biasa, Pangsit, Ceker, Bakso, Komplit, Sayap, Telur Puyuh, Tulangan, Kepala, Pangsit Kuah
- **Minuman** (5 items): Teh Anget, Es Teh Manis, Es Jeruk Peras, Es Nutrisari, Es Tawar
- **Topping Tambahan** (8 items): Ceker, Pangsit Rebus, Bakso, Telur Puyuh, Kepala Ayam, Sayap, Tulangan, Pangsit Goreng

### Packages (4 packages)
1. **Paket Lengkap** — Mie Ayam Komplit + Es Teh Manis
2. **Paket Favorit** — Mie Ayam Pangsit + Es Teh Manis
3. **Paket Kenyang** — Mie Ayam Bakso + Es Teh Manis
4. **Paket Spesial** — Custom package

### Status Values
- `available` - Item is in stock
- `limited` - Item has limited stock
- `sold_out` - Item is out of stock

## 8. Services / Modules

### FirebaseService (js/firebase-config.js)
**Purpose**: Firebase initialization and data management (menu, packages, stock)

**Public API**:
```javascript
window.FirebaseService = {
  config,                    // Firebase config object
  db,                        // Firebase database reference
  menuRef,                   // Menu data reference
  packagesRef,               // Packages data reference
  INITIAL_MENU_DATA,         // Default menu data (24 items)
  INITIAL_PACKAGES_DATA,     // Default packages data (4 packages)
  
  // Menu functions
  seedInitialMenu(),         // Seed initial menu data
  onMenuItemChange(),        // Listen to single item changes
  onAllMenuChange(),         // Listen to all menu changes
  updateMenuItem(),          // Update menu item
  deleteMenuItem(),          // Delete menu item
  getMenuItem(),             // Get single menu item
  getAllMenu(),              // Get all menu items
  nameToSlug(),              // Convert name to URL slug
  isValidStatus(),           // Validate status string
  getStatusText(),           // Get status display text
  
  // Packages functions
  seedInitialPackages(),     // Seed initial packages data
  addPackage(),              // Add new package
  getPackages(),             // Get all packages
  updatePackage(),           // Update package
  deletePackage(),           // Delete package
  onAllPackagesChange(),     // Listen to all packages changes
  
  // Backward compatibility aliases
  stockRef, seedInitialStock(), onStockChange(), onAllStockChange(),
  updateStock(), getStockStatus(), getAllStock()
};
```
