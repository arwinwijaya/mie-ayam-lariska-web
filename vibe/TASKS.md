## What we're working with

Mie Ayam Lariska Web adalah website statis untuk restoran mie ayam di Indonesia. Website ini menyediakan menu lengkap dengan 24 item dalam 3 kategori, 4 paket rekomendasi, sistem pemesanan via WhatsApp, dan dashboard admin lengkap untuk manajemen menu dan paket. Menggunakan Firebase Realtime Database untuk penyimpanan data dan Firebase Hosting untuk deployment.

**Stack**: Vanilla JS + Firebase Realtime Database + Firebase Hosting
**Pattern**: Static HTML with IIFE JavaScript modules
**Files**: ~80+ files
**Test Framework**: Playwright (E2E)
